# Kuantum Kaos Yönetimi

Bu proje dört farklı dilde uygulanmıştır:

- C#
- Java
- Python
- JavaScript (Node.js)

Amaç:
Kuantum veri deposundaki nesneleri yönetmek, analiz etmek ve stabiliteleri sıfır seviyesine düşerse özel bir hata (KuantumCokusuException) fırlatarak sistemi sonlandırmaktır.

Her dilde:
- Soyut sınıf (abstract)
- Arayüz (interface)
- Kalıtım (inheritance)
- Polimorfizm (polymorphism)
- Özel hata sınıfı (exception)
- Menü tabanlı oyun döngüsü

bulunmaktadır.

Kodlar sınıflar arası yapıyı koruyacak şekilde sadeleştirilmiş ve anlaşılır biçimde yazılmıştır.
